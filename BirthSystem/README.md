# BirthSystem

출생 시스템 플러그인입니다.

## 기능
- 첫 로그인 시 랜덤 차원 선정 (가중치 기반)
- 낮은 확률로 마계/천계/선계/요계/명계/정령계 출생
- 출생 차원에 따른 종족 결정

## 종족 목록
| 차원 | 종족 | 특성 |
|------|------|------|
| 마계 | 악마 | STR+3, 마기+5, 성장률 ×1.5 |
| 천계 | 천사 | WIS+3, 천기+5, 성장률 ×1.5 |
| 선계 | 선인 | 전 스탯+2, 성장률 ×2.0 |
| 요계 | 요괴 | DEX+3, 요기+5, 성장률 ×1.5 |
| 명계 | 영혼체 | INT+3, 혼기+5, 성장률 ×1.5, 사망 시 드롭 없음 |
| 정령계 | 정령인 | INT+2, 정령 마나+5, 성장률 ×1.5 |
| 판타지/무협/지구 | 인간 | 기본 스탯, 성장률 ×1.0 |

## 의존성
- DimensionCore

## 설정 (config.yml)
- `birth.dimensions`: 각 차원별 가중치 (합산 확률로 계산)
- `races.<차원>.name`: 종족 이름
- `races.<차원>.bonuses`: 스탯 보너스
- `races.<차원>.growth_multiplier`: 성장률 배율
- `races.<차원>.no_death_drop`: 사망 시 드롭 없음 여부
