# JobClassSystem

직업/문파/헌터 시스템 플러그인입니다.

## 기능

### 판타지 직업 (17종)
검전사, 성기사, 마법사, 룬공학자, 정령주술사, 음유시인, 아케인아처, 기갑창병, 점성술사, 혈귀술사, 드루이드, 네크로맨서, 원소술사, 연금술사, 비전학자, 암살자, 수도사

### 무협 무공 체계
- 무공 종류: 외공, 내공, 경신, 심법, 경공
- 무공 학습 시 다른 무공 학습 확률 감소 (육체적 특성)
- **이름 미정 무공**: 서버에 5개만 존재, 모든 무공 학습 가능하게 함

### 지구 헌터 체계
- 랭크: E → D → C → B → A → S → SS → SSS
- 각성 계열 (변경 불가): 섀도우, 염동력, 강화, 소환, 중력, 전자기, 시간왜곡, 생체개조, 데이터해킹

## 명령어
| 명령어 | 설명 |
|--------|------|
| `/job info` | 직업 정보 조회 |
| `/job select <직업>` | 판타지 직업 선택 |
