# DeathRealm

명계 & 부활 시스템 플러그인입니다.

## 기능
- 사망 시 명계(3차원 공통 허브)로 이동
- 부활 방법:
  1. 부활 퀘스트 완료
  2. 소울 코인 지불 (신전에서)
  3. 미리 소울 코인 예치 (즉시 부활)
  4. 주변 인물이 대신 비용 지불
- 부활 비용: 기본 50,000 소울 (AI NPC에 의해 인상 가능)
- 명계 출생 영혼체 종족: 사망 시 드롭 없음
- 명계 구성: 왕도, 계약의 방, 시장, 검문소
- 명계 내 PVP 금지, 소울 코인으로만 거래

## 명령어
| 명령어 | 설명 |
|--------|------|
| `/resurrect` | 소울 뱅크 예치금으로 부활 시도 |
| `/soulbank deposit <금액>` | 부활 비용 미리 납부 |
| `/soulbank withdraw <금액>` | 예치금 인출 |
| `/soulbank balance` | 예치금 조회 |
