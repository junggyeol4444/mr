# TowerGate

탑 & 게이트 시스템 플러그인 (지구 차원)입니다.

## 기능
- 100층 탑 구조 (층별 테마)
- 특수 층: 시간제한층, 금기층, 국가 협력층
- 개인 등반 기록 및 랭킹
- 국가 단위 랭킹
- 50층 이상 등반 시 싱크 게이트 이벤트 트리거
- 게이트: 이미 등반한 층 기반 난이도 선택 (EASY/NORMAL/HARD/EXTREME)

## 층별 테마
| 층 범위 | 테마 |
|---------|------|
| 1~10 | 생존 |
| 11~20 | 퍼즐 |
| 21~30 | 환경 |
| 31~40 | 혼합 |
| 41~50 | 국가 협력 |
| 51~60 | 심층 생존 |
| 61~70 | 고급 퍼즐 |
| 71~80 | 극한 환경 |
| 81~90 | 탑 보스 |
| 91~100 | 심연의 문 |

## 명령어
| 명령어 | 설명 |
|--------|------|
| `/tower info` | 내 탑 기록 조회 |
| `/tower floor <층>` | 층 정보 조회 |
| `/tower rank` | 개인 랭킹 조회 |
