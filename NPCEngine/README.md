# NPCEngine

AI NPC 시스템 플러그인입니다.

## 기능
- AI 자율 사고 NPC
- 호감도/적대도 시스템 (-100 ~ +100)
  - 50 이상: 동맹 (전용 퀘스트 해금)
  - -50 이하: 적대
- NPC 간 정치/전쟁/상거래
- 플레이어와 동맹/적대 가능
- 고유 서사 보유 NPC
- 죽은 NPC는 명계에서 생활
