# RaceEvolution

종족 진화 & 초월 시스템 플러그인입니다.

## 기능
- 종족 진화 (되돌릴 수 없음): 정령인, 반마족, 기계인, 타락자
- 종족 초월: 스탯/스킬 한도 초과 시 신적 존재로 승격
- 초월자에게서 전수 가능 (전해주는 측은 2배 지출)

## 명령어
| 명령어 | 설명 |
|--------|------|
| `/evolve <경로>` | 진화 경로 선택 (확인 필요) |
| `/evolve confirm <경로>` | 진화 확정 |
| `/transcend` | 종족 초월 |

## 의존성
- DimensionCore, BirthSystem, StatSystem
