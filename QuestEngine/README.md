# QuestEngine

퀘스트 엔진 플러그인입니다.

## 기능
- 5가지 퀘스트 유형 지원
  - 개인 퀘스트: 성장 중심
  - 자유(서브) 퀘스트: 탐험/평판/이벤트
  - 직업/문파 퀘스트: 고유 기술 해금
  - 메인 퀘스트: 대륙 단위 서사
  - 게이트 이벤트 (지구 차원)
- 퀘스트 진행 상태 추적
- 보상 시스템 (경험치)

## 명령어
| 명령어 | 설명 |
|--------|------|
| `/quest list` | 진행 중인 퀘스트 목록 |
| `/quest info <ID>` | 퀘스트 상세 정보 |
| `/quest accept <ID>` | 퀘스트 수락 |
| `/quest abandon <ID>` | 퀘스트 포기 |

## 기본 퀘스트
- `tutorial_combat`: 첫 전투 (개인)
- `explore_world`: 세계 탐험 (자유)
- `main_lumen`: 루멘의 불씨를 찾아서 (메인)
- `gate_event_001`: 게이트 토벌 (게이트 이벤트)
