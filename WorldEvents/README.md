# WorldEvents

월드 이벤트 플러그인입니다.

## 기능
- 4가지 주요 월드 이벤트
  - **마계 침식 주간**: 판타지 대륙 디버프, 마물 및 악마 출현
  - **천계 성물강림**: 타락한 성물 봉인 해제, 월드보스 출현
  - **요계 개문**: 드롭률 상승, 요괴 토벌 및 포획 가능
  - **싱크 게이트**: 탑 50층 이상 등반 후 글로벌 이벤트 발동
- 차원 균형 변동 연동 이벤트 트리거
- 이벤트 지속 시간 관리 및 종료 알림

## 이벤트 트리거 조건
| 이벤트 | 트리거 조건 |
|--------|------------|
| 마계 침식 주간 | 차원 균형 30 이하 |
| 천계 성물강림 | 차원 균형 71 이상 |
| 요계 개문 | 관리자/스케줄 설정 |
| 싱크 게이트 | 탑 50층 이상 등반자 존재 |
