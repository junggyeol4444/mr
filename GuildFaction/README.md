# GuildFaction

길드 & 세력 시스템 플러그인입니다.

## 기능
- 길드 창설 및 관리
- 최대 인원: 50명
- 영지전: 도시 점령, 세금 부과 (1~20%)
- 명계 연합체 구성
- 국가 길드전 (지구 차원)
- PVP 리그

## 길드 직위
| 직위 | 설명 |
|------|------|
| 단원 | 일반 길드원 |
| 간부 | 관리 권한 |
| 부길드장 | 부관리자 |
| 길드장 | 길드 창설자/관리자 |

## 명령어
| 명령어 | 설명 |
|--------|------|
| `/guild create <이름>` | 길드 창설 |
| `/guild info` | 길드 정보 조회 |
| `/guild leave` | 길드 탈퇴 |
| `/guild disband` | 길드 해산 (길드장 전용) |
